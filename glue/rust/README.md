<h1>warduino</h1>

<p><strong>Crate for accessing WARDuino primitives in Rust.</strong></p>

</div>

## Features

+ Access WARDuino primitives from Rust
+ Use Rust types and structs for arguments

## Installation

Add the following to your Cargo.toml:

```toml
[dependencies]
warduino = "0.1.0"
```
