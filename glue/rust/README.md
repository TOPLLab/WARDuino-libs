<h1>rs-warduino</h1>

<p><strong>Crate for accessing WARDuino primitives in Rust.</strong></p>

</div>

## Features

+ basic I/O constants
+ time primitives
+ serial primitives
+ digital I/O primitives
+ Wi-Fi primitives
+ HTTP primitives with simple structs for options
+ interrupt primitives
+ MQTT primitives

## Installation

This package requires **WARDuino 0.2.3+**.
